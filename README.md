# Netflix Excel Dashboard

## Overview

This project presents an interactive Netflix Dashboard built using Microsoft Excel. The dashboard analyzes Netflix content and provides insights into ratings, countries, release years, and content types.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning

Dashboard Features

* Rating Analysis
* Top 10 Countries by Content
* Top 10 Countries by Type
* Release Year Analysis
* India vs USA Comparison
* Movies vs TV Shows Distribution
* Interactive Filters

Dataset

Netflix Movies and TV Shows Dataset

Key Insights

* Movies make up the majority of Netflix content.
* The United States has the highest content contribution.
* Most content was released after 2015.
* TV-MA is among the most common ratings.
